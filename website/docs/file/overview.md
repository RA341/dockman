# File Explorer

Dockman includes a full-featured file explorer for managing your Docker compose files, configs and more.

## Features

- **Basic operations** — Create, delete, rename, and move files
- **Upload/Download** — Drag and drop files from you computer, or download them
- **Quick search** — Jump to any file instantly with the search bar
- **File aliases** — Browse directories outside your compose root (like `/mnt/media` or `/opt/appdata`)
- **SQLite browser** — View and edit `.db` files with a built-in GUI
- **YAML formatting** — Auto-format compose files with `Alt + L`
